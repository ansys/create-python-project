Module Documentation¶

Create-Python-Project is a command-line tool so it shouldn’t normally be imported. However, it is MIT-licensed and if you wish to develop using it you can view the auto-generated documentation here!

The package contains two primary modules: main.py and generator.py.

generator.py¶

This contains most of the package’s logic and covers template generation.

class src.generator.ProjectTemplateAndDestinationChecker(template, destination)¶

Bases: object

Provides capability to check validity of a supplied template and destination.

Given a template and destination, this class provides the power to check the validity of both. Once initialised, the checks can be applied at a granular level or all at once using the check() method.

Parameters

  • template (ProjectTemplate) – pathlib.Path to the template directory

  • destination (Path) – pathlib.Path to the destination directory

src.main.cli(root_folder=None)¶

Command Line Interface function.

Constructs the CLI and executes the program itself. The root folder is the one in which the project should be created. This defaults to the current working directory but can be set to be something else. This functionality exists primarily for testing purposes.
