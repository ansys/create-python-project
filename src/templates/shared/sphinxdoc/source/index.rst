Welcome to $project_name's documentation!
========================================================================================================================

Welcome to the documentation template for $project_name.

In order to build this documentation all you should need to do is:

1. Install the requirements in ``requirements_doc.txt``
2. ``cd`` into the ``$project_name/doc/`` directory
3. Run ``make html`` (on Linux) or ``make.bat html`` (on Windows)
4. View ``$project_name/doc/build/html/`` to see the built docs.
